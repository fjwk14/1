# 意思決定 (decide)

選択を、感覚ではなく「基準に照らした比較」に変える。最後は逃げずに**推奨を1つ**出す。

## 手順

1. **決めることを言い直す** — 何を選ぶ問題なのかを1文で。
2. **「良い」の定義を握る** — 判断基準を挙げ、重要度(重み)を軽く付ける。例: コスト / 速さ / 可逆性 / 気持ちの納得感。基準がズレると比較が無意味になるのでここを丁寧に。
3. **選択肢を並べる** — 見落としがちな「今のまま何もしない」も含める。
4. **基準 × 選択肢で比較** — 表にする。各セルは短い根拠付きで。
5. **推奨を出す** — どれを推すか、なぜか、そして「**何が分かれば結論が変わるか**」を添える。
6. **間違えたときのコストを見る** — 可逆な決定(後で戻せる)は速く決めてよい。不可逆で高コストなら、確認や情報収集を勧める。

## 原則

- 中立な整理で終わらせない。**立場を取る**(ただし根拠と反証条件を明示)。
- 本人の価値観に関わる決定(キャリア・人間関係など)は、基準の言語化を手伝い、最終判断は本人に委ねる。
- 情報が足りず決められないなら、「決める前に確かめる1つ」を示す。
