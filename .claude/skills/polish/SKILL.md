# 推敲 (polish)

文章を、意味と書き手の個性を保ったまま、目的の相手に届く形に磨く。原文を勝手に別人の文体に置き換えない。

## 手順

1. **前提を掴む** — 読み手は誰か / 目的は何か(依頼・謝罪・共有・説得)/ 望むトーン(丁寧・カジュアル・簡潔)。不明なら1つだけ短く確認、文脈から明らかならそのまま進む。
2. **推敲版を出す** — 直した本文を先に提示する。
3. **変更点を短く説明** — 「何を・なぜ」変えたかを3〜6個。冗長の削除 / 曖昧さの解消 / トーン調整 / 誤り訂正、のように理由付きで。
4. **必要なら別トーン版** — もっと硬い/柔らかい版が有用そうなら1つ添える。

## 原則

- **意味を変えない。** 元の主張・ニュアンス・敬意のレベルを保つ。
- **過剰に整えない。** 元がカジュアルなら整いすぎた敬語にしない。書き手の癖のうち「味」になっているものは残す。
- **事実・約束・数字**を触るときは必ず明示して確認を促す(勝手に締切や条件を変えない)。
- 外国語の文なら、自然さと文法の両方を直し、意図が変わる修正は注記する。
