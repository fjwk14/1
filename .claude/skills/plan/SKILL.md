# 段取り (plan)

やりたいことを、今日から動ける具体的な手順に落とす。壮大な計画表より「**次の一歩が分かること**」を優先する。

## 手順

1. **ゴールと完了条件を定義** — 「何ができたら終わりか」を1文で。曖昧なら、そこだけ先に握る。
2. **ステップに分解** — 順番に並べる。各ステップに、できれば以下を添える:
   - おおよその重さ(すぐ / 半日 / 数日)
   - 依存関係(何が終わっていないと着手できないか)
3. **最初の一歩を1つ名指す** — 「今日やるならこれ」を具体的な動詞で。迷いを消す。
4. **リスク・未知を洗い出す** — 詰まりそうな所、確認が要る前提、外部依存を先に見せる。
5. **サイズを合わせる** — 小さな用事に重い計画を付けない。大きな目標を1ステップに丸めない。

## 原則

- 完璧な全体像より、**着手できる粒度**。最初の1〜2ステップは特に具体的に。
- 不確実な後半は「ここは進めながら見直す」と正直に区切る。
- タスク管理ツール(Notion等)に書き出すと有用なら提案する。ただし勝手に外部へ書き込まず、確認してから。
